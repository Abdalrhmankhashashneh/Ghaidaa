HTML

Stands for : HyperText Markup Languge

Tim Berners-Lee

is an English computer scientist best known as the inventor of the World Wide Web

Berners-Lee is the director of the World Wide Web Consortium (W3C), which oversees the continued development of the Web.

Version

  • HTML 1.0: released in 1991

  • HTML 2.0: released in 1995

  • HTML 3.2: released in 1997

  • HTML 4.01: released in 1999

  • XHTML: released in 2000

  • HTML5: released in 2014

CSS

Stands for : Cascading Style Sheets.

- Cascading refers to the way CSS applies one style on top of another.
- Style Sheets control the look and feel of web documents.


HTML work hand in hand:

- HTML sorts out the page structure.
- CSS defines how HTML elements are displayed.

CSS History ?

CSS has gone through 3 phases so far :

  • CSS 1

    The first CSS specification to become an official Recommendation is CSS level 1, published on December, 1996.

  • CSS 2-2.1

    CSS 2 was developed published as a recommendation in May 1998.
    CSS 2.1 first became a Candidate Recommendation on 2004.

  • CSS 3

    Work on CSS level 3 started around the time of publication of the original CSS 2 recommendation. The earliest CSS 3 drafts were published in June 1999.
